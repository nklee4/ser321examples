You can make the following GET requests:

  • /file/sample.html
     Purpose--> returns the content of the file sample.html

  • /json
     Purpose--> returns a json of the /random request

  • /random
     Purpose--> returns index.html

  • /multiply?num1=x1&num2=x2
     Arguments--> x1 and x2 represents a valid integer
      Purpose--> returns the multiplicative solution of num1 and num2

  • /github?query=users/amehlhase316/repos
     Purpose--> returns full_name and id of repos, and login of the owner of each repo

  • /cone?radius=x1&height=x2
     Arguments--> x1 and x2 represents a valid double
      Purpose--> returns the volume of a cone based on input radius and height

  • /sentence?noun1=y1&noun2=y2
     Arguments--> y1 and y2 represents a valid word
      Purpose--> returns a random simple sentence based on input nouns
